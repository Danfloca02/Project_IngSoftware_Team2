CLASES:
 
Usuario (Clase Base)
    Atributos:
        ID
        Nombre
        Email institucional
        Rol (Enum: Estudiante/Profesor/PersonalApoyo)
        Foto
        Estado (activo/inactivo)
    Relaciones:
        Crea múltiples Publicaciones (0..*)
        Participa en múltiples Eventos (0..*)
        Recibe múltiples Notificaciones (0..*)
        Revisa Calendaario (1)

Moderador (Hereda de Usuario)
    Atributos:
        Nivel de permisos
        Área de moderación
    Responsabilidades:
        Aprobar/rechazar publicaciones
        Moderar comentarios
        Gestionar eventos

Estudiante (Hereda de Usuario)
    Relaciones:
        Pertenece a una Facultad (1)
        Cursa una o dos Carreras (1..2)

Profesor (Hereda de Usuario)
    Relaciones:
        Crea eventos

PersonalApoyo (Hereda de Usuario)
    Relaciones:
        Realizar Tareas de Apoyo


Facultad
    Atributos:
        ID
        Nombre
        Descripción
        Ubicación
    Relaciones:
        Tiene múltiples Carreras (n)
        Contiene múltiples Usuarios (*)
        Alberga múltiples Espacios (*)

Carrera
    Atributos:
        ID
        Nombre
        Descripción
        Coordinador
    Relaciones:
        Pertenece a una Facultad (1)
        Tiene múltiples Usuarios (*)

Evento
    Atributos:
        ID
        Título
        Descripción
        Fecha y hora
        Tipo (académico/cultural/deportivo)
        Estado (pendiente/aprobado/rechazado)
    Relaciones:
        Creado por Usuario (1..*)
        Se realiza en Espacio (1)
        Tiene múltiples Participantes (*)
        Genera Notificaciones (1..*)

Notificacion
    Atributos:
        ID
        Contenido
        Fecha
        Tipo
        Estado (leída/no leída)
    Relaciones:
        Destinada a Usuario (*)
        Relacionada con Evento/Publicación (1)

Espacio
    Atributos:
        ID
        Nombre
        Tipo (aula/auditorio/área común)
        Capacidad
        Disponibilidad
    Relaciones:
        Pertenece a Facultad (1..1)
        Alberga múltiples Eventos (1..*)

Publicacion
    Atributos:
        ID
        Título
        Contenido
        Fecha
        Estado (pendiente/aprobado/rechazado)
        Tipo (académica/informativa/social)
    Relaciones:
        Creada por Usuario (1)
        Tiene múltiples Comentarios (0..*)
        Genera Notificaciones (0..*)

Comentario
    Atributos:
        ID
        Contenido
        Fecha
        Estado
    Relaciones:
        Pertenece a Publicación (1)
        Creado por Usuario (1)

Calendario
    Relaciones:
        Guarda Eventos (0..*)
        Muestra Eventos (0..*)
